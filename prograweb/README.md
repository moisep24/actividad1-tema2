Actividad 1 - Tema 2: Ejercicios de Programación Web
Este repositorio contiene los 10 ejercicios prácticos requeridos para la primera actividad del Tema 2 de la materia.

Organización de los archivos
Raíz: Aquí está el menú principal (index.html) para abrir y navegar por cada uno de los ejercicios.
css: Carpeta con los archivos de diseño y hojas de estilo de las páginas.
js: Carpeta con los scripts de JavaScript para el control de la lógica, validación de formularios y manejo del DOM.
img: Imágenes locales utilizadas en el desarrollo de las prácticas.
Conceptos aplicados
Maquetado y estructura de páginas usando HTML5.
Estilos personalizados y diseño limpio con CSS.
